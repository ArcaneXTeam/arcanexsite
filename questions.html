Frequently Asked Questions

Find answers to common questions about ArcaneX

What is a Roblox executor?

A Roblox executor (sometimes called a roblox injector or lua executor) is a desktop application that runs custom Lua scripts in a Roblox session. Developers use them to automate gameplay, prototype game ideas and learn Lua. ArcaneX is one such executor, focused on a fast attach engine and a clean editor.

What is the difference between a Roblox injector and an executor?

In community jargon the terms are used almost interchangeably. Strictly speaking, an injector is the component that attaches to the Roblox process, while the executor is the Lua runtime that runs your scripts. Most modern tools — including ArcaneX — bundle both in one application.

Is ArcaneX really free?

Yes. ArcaneX is free to download and use — no hidden fees, no subscriptions, no key system, no ads.

What operating systems does ArcaneX support?

ArcaneX currently supports Windows 10 and Windows 11 (64-bit). Make sure your system has the latest updates and a recent version of the .NET runtime for best results.

Do I need a key to use ArcaneX?

No. ArcaneX is fully keyless — no link‑bypassers, no daily verification, no waiting. Open it and you're ready to script.

How do I install ArcaneX?

Download the ZIP from the homepage, extract it to a folder you control and run ArcaneX.exe. No installer required.

Can I use ArcaneX on multiple devices?

Yes. There are no device limits — copy the folder to any Windows 10/11 PC you own and run it.

What scripts are compatible with ArcaneX?

ArcaneX supports standard Lua scripts and most community APIs. Popular utilities like Infinite Yield and Dark Dex work out of the box, and the built-in Script Hub gives quick access to a curated list.

How often is ArcaneX updated?

We update ArcaneX regularly to keep it compatible with the latest Roblox client builds and to add features. Updates are delivered automatically — you'll see a prompt the next time you launch the app.

Is using a Roblox executor allowed?

Running third‑party scripts in Roblox may conflict with Roblox's Terms of Service. ArcaneX is provided as a developer and learning tool — use it responsibly, prefer single-player or private servers for experiments, and respect the rules of any game you play.
